
# Duplo

A self-hosted deployment platform buildt on k3s.




## Running with docker:

```bash
  docker run -d herjuus/duplo -p 3000:3000
```

## Run the API Locally
*(You also need k3s on the same server)*

Clone the project

```bash
  git clone https://github.com/Herjuus/Duplo.git
```

Go to the project directory

```bash
  cd duplo/api
```

Start the server

```bash
  cargo run
```
